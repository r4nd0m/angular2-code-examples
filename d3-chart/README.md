# Task

Implement a progress bar component using D3 library.

# Setup Instructions

0. Install node, npm and git
1. Open command prompt (bash, cmd etc.)
1. git clone https://github.com/r4nd0m/code-examples
2. cd code-examples/d3-chart
3. npm install
4. npm start
